A multi-threaded HTTP proxy server built on top of Boost's Asio.

### Running

```
git clone https://github.com/Tamashii90/http-proxy-cpp
cd http-proxy-cpp
make run
```

Don't forget to change your proxy settings. Boost Asio is included for convenience.
